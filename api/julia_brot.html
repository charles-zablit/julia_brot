julia_brot.julia_set.is_in_julia(z: complex, c: complex, max_iter: int = 10) → bool[source]#

Checks if a point is in a given Julia set for a maximum number of iterations.

Parameters:

  • z (complex) – Point to check for.

  • c (complex) – Constant value of the Julia set.

  • max_iter (int) – Maximum number of iterations.

Returns:

True if the point is in the set, False otherwise.

Return type:

bool

Example

>>> is_in_julia(z=0.25 + 0.25j, c=0.25)
True
>>> is_in_julia(z=0.25 + 0.25j, c=0.25, max_iter=100)
True
julia_brot.julia_set.plot_julia(c: complex = 0j, zmin: complex = -2 - 2j, zmax: complex = 2 + 2j, pixel_size: float = 0.1, max_iter: int = 10, figname: str | bytes | PathLike | None = None)[source]#

Plot the Julia set corresponding to the given c constant.

Parameters:

  • c (complex) – Julia constant.

  • zmin (complex) – Lower bound limit.

  • zmax (complex) – Upper bound limit.

  • pixel_size (float) – Pixel size of the resulting plot.

  • max_iter (int) – Maximum number of iterations.

  • figname (Optional[Union[str, bytes, os.PathLike]]) – Optional path to save the image to. Will not be saved if None.

Example

>>> plot_julia()
None
>>> plot_julia(c=-0.8 + 0.156j, zmin=-2-1j,
               zmax=2+1j,
               pixel_size=5e-4,
               max_iter=100, figname=figname)
None
